# Getting Started

Welcome to FreeSTAR Everywhere!  
This guide helps you set up and begin using the platform.

## Installation

1. Clone the repository:
    ```
    git clone https://github.com/FreeSTAR-Network/freestar-everywhere-wiki.git
    ```
2. Install dependencies:
    ```
    pip install -r requirements.txt
    ```
3. Serve locally:
    ```
    mkdocs serve
    ```
4. View your wiki at [localhost:8000](http://localhost:8000)
